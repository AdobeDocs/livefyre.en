# Switch Networks Using the Livefyre Mobile App

>1. Log in to the Livefyre mobile app.
>   
>1. Select `uicontrol Menu &gt; Settings`.
>   
>1. Select the network to use.
>   
>1. Select `uicontrol Save`.
>
