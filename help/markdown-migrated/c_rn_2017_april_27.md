    <td>Storify 2</td>

   </tr> 
   <tr> 
    <td>Bug</td> 
    <td>Streams</td> 
    <td>Fixed a bug that prevented rules from being copied over to a new stream when cloning an existing stream.</td> 
   </tr> 
   <tr> 
    <td>Bug</td> 
    <td>Users</td>
