# Create a Story

>1. Click `uicontrol New App`.
>   
>1. Click `uicontrol Storify 2`.
>   
>1. Enter an `uicontrol App Title`.
>   
>1. Select a `uicontrol Site`.
>   
>1. Click `uicontrol Create`.
>   
>
