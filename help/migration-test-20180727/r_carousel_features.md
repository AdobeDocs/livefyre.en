# Carousel Features




The following features are available in a Carousel:

* SAFE Filter
* Social Sharing
* Media Expansion
* Upload Button Compatible
* CSS Styling
* Custom Text Strings
* Twitter Action Buttons
* Date and Timestamp
* Avatars
* Default Styling
* Items to load
* Translation sets
