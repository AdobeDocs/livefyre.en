# Filmstrip Features{#filmstrip-features}

Learn the features in a Filmstrip App.

The Filmstrip App has the following features:

* Real-Time Streaming
* SAFE Filter 
* Social Sharing
* Media Expansion
* Permalinks
* CSS Styling
* Twitter Action Buttons
* Date and Timestamp
* Default styling
* UGC Commerce
