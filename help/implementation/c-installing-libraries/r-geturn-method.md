# getUrn Network Method{#geturn-network-method}

This method returns the URN for this network.

## Java Example {#section_nyl_ycs_rz}

```
network.getUrn(); 

```

Sample output:

```
"urn:livefyre:network=`example.fyre.co`" 

```

## NodeJS Example {#section_xkd_gds_rz}

```

```

Sample output:

```

```
