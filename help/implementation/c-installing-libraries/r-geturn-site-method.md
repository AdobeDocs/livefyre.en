# getUrn Site Method{#geturn-site-method}

This method returns the URN for this site.

## Java Example {#section_nyl_ycs_rz}

Usage:

```
site.getUrn();
```

Sample output:

```
"urn:livefyre:network=`example.fyre.co`:site=1" 

```
